# AI Resume Screener

An AI-powered resume screening tool that matches resumes 
with job descriptions using Groq LLM and Streamlit.

## Features
- Upload resume in PDF format
- Paste job description
- Get Match Score, Missing Skills, Suggestions and Final Verdict

## Tech Stack
- Python
- Groq API (LLaMA 3.3)
- Streamlit
- PyPDF2

## How to Run
1. Install dependencies: pip install -r requirements.txt
2. Add your GROQ_API_KEY in .env file
3. Run: streamlit run app.py
